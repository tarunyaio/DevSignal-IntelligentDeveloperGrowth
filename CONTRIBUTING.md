## 🛠️ Getting Started in 5 Minutes

### 1. The Setup
You'll need Node.js installed. We use a high-performance stack:
```bash
# Clone your fork
git clone https://github.com/YOUR_USERNAME/DevSignal-IntelligentDeveloperGrowth.git
cd DevSignal-IntelligentDeveloperGrowth

# Install dependencies (Frontend)
npm install

# Build the vibes
npm run dev
```

### 2. Finding Your First Task
Look for issues labeled `good first issue` or `visual enhancement`. If you have a wild idea for a new feature, open a "Feature Request" issue first so we can chat about it!

---

## 📮 How to Submit Your Magic

1. **Branch it out**: `git checkout -b feature/your-cool-idea`
2. **Commit with heart**: We follow professional commit standards, but keep it human. 
   - *Example*: `feat: add glowing border to repo cards`
3. **The PR Dance**: When you open a Pull Request, tell us the *why* and show us the *how* (screenshots/recordings are LOVED here).

---

## ✅ The Golden Rules
- **Respect the Design**: We use `framer-motion` for animations and `lucide-react` for icons. Keep the premium feel!
- **Clean Code, Happy Life**: Follow the TypeScript patterns already in the codebase.
- **Be Kind**: We are a community of builders. Toxicity is the only bug we don't allow.
